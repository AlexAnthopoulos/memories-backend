# memories-backend
Memories Project Back-End.
 This is REQUIRED to run together with Memories Front end to view the full app
 
 ## Technologies used
 
 - MongoDB
 - NodeJS
 - Express
 - Mongoose
 - BcryptJS
 - Passport
 
 ## Instal dependencies
 
 ```
 yarn
 ````

## Server running
```
localhost:5000
```
